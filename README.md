# Prácticas de Python

Repositorio con ejercicios y prácticas desarrolladas durante mi formación académica en programación con Python.

## Objetivo

Reforzar fundamentos de programación, lógica computacional, estructuras de control, manejo de datos y desarrollo de pequeños programas interactivos utilizando Python.

## Contenido del repositorio

### Fundamentos

Ejercicios orientados a los conceptos básicos del lenguaje Python:

- Variables y tipos de datos
- Cadenas de texto
- Formateo de strings
- Métodos de strings
- Secuencias de escape
- Operadores numéricos
- Conversión de tipos

### Ejercicios

Programas prácticos para reforzar la lógica de programación:

- Calculadora básica
- Calculadora interactiva con bucle de salida
- Ejercicios con condicionales
- Ejercicios con ciclos

### Manejo de archivos

Sección destinada a prácticas relacionadas con lectura, escritura y manipulación de archivos usando Python.

## Tecnologías utilizadas

- Python
- Visual Studio Code
- Git
- GitHub

## Aprendizajes

A través de estos ejercicios he reforzado conceptos fundamentales de programación, pensamiento lógico, resolución de problemas y buenas prácticas iniciales en el uso de GitHub como portafolio técnico.
